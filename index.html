üîß API Usage

You can also use this service directly via API calls:

GET /api/index?url=IMAGE_URL&w=WIDTH&q=QUALITY&l=GRAYSCALE
  • url - Source image URL (required)
  • w - Maximum width in pixels (required, 1-8192)
  • q - Quality percentage (required, 10-100)
  • l - Grayscale mode (optional, 1 for grayscale)
